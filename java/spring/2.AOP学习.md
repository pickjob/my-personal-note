# AOP学习
- 相关接口和类
    ```java
    // Advice: 通知, 标志接口
    //      - Interceptor: 拦截器, 标志接口
    //          - MethodInterceptor: invoke
    //          - ConstructorInterceptor: construct
    //      - BeforeAdvice: 标志接口
    //          - MethodBeforeAdvice: before
    //      - AfterAdvice: 标志接口
    //          - AfterReturningAdvice: afterreturn
    //          - ThrowsAdvice: 标志接口
    //      - DynamicIntroductionAdvice
    public interface MethodInterceptor extends Interceptor {
        Object invoke(MethodInvocation invocation) throws Throwable;
    }
    public interface ConstructorInterceptor extends Interceptor  {
    	Object construct(ConstructorInvocation invocation) throws Throwable;
    }
    public interface MethodBeforeAdvice extends BeforeAdvice {
        void before(Method method, Object[] args, @Nullable Object target) throws Throwable;
    }
    public interface AfterReturningAdvice extends AfterAdvice {
        void afterReturning(@Nullable Object returnValue, Method method, Object[] args, @Nullable Object target) throws Throwable;
    }
    public interface DynamicIntroductionAdvice extends Advice {
        boolean implementsInterface(Class<?> intf);
    }
    // Pointcut
    public interface Pointcut {
        ClassFilter getClassFilter();
        MethodMatcher getMethodMatcher();
        Pointcut TRUE = TruePointcut.INSTANCE;
    }
    // Advisor: Spring AOP主要保存Advice代理信息
    //      - PointcutAdvisor
    //          - StaticMethodMatcherPointcutAdvisor(静态方法匹配)
    //          - InstantiationModelAwarePointcutAdvisor(延迟加载接口)
    //      - IntroductionAdvisor
    public interface Advisor {
        Advice EMPTY_ADVICE = new Advice() {};
        Advice getAdvice();
        boolean isPerInstance();
    }
    public interface PointcutAdvisor extends Advisor {
	    Pointcut getPointcut();
    }
    // PointcutAdvisor:
    //      - StaticMethodMatcherPointcutAdvisor: 按静态方法PointcutAdvisor类
    //      - InstantiationModelAwarePointcutAdvisor: 是否懒加载初始化接口
    //      - AbstractPointcutAdvisor: 抽象PointcutAdvisor类
    //          - AbstractBeanFactoryPointcutAdvisor: 抽象实现BeanFactoryAware类

    // ProxyProcessorSupport: 代理类创建基础, 本质是BeanPostProcessor实现功能
    //      - AbstractAdvisingBeanPostProcessor: 指定Advisor代理类
    //          - AbstractBeanFactoryAwareAdvisingPostProcessor
    //              - MethodValidationPostProcessor
    //      - AbstractAutoProxyCreator: 自动代理同一类
    //          - AbstractAdvisorAutoProxyCreator: 自动代理Advisor类
    //              - DefaultAdvisorAutoProxyCreator: 默认自动代理Advisor接口
    //              - AspectJAwareAdvisorAutoProxyCreator / AnnotationAwareAspectJAutoProxyCreator: 自动给ApsectJ类代理
    //              - InfrastructureAdvisorAutoProxyCreator: 框架基础代理对象创建
    ```
