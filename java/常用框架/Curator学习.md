# Curator学习
- Retry Policies
    - RetryOneTime: 重新尝试一次
    - RetryNTimes: 重新尝试N次
    - RetryUntilElapsed: 直到过了重新尝试
    - ExponentialBackoffRetry: 递增延迟重新尝试

    - transactionOp()
    - transaction()
    - getACL()
    - setACL()
    - getConfig()

    - reconfig()
