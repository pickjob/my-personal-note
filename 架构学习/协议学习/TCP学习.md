# TCP协议
- 端口号类型
  - 数值端口号(0-1023)
    - udp
      - 53 : 域名服务
      - 67/68 : DHCP
      - 69 : TFTP
      - 111 : RPC
      - 123 : NTP
      - 161/162 : SNMP
      - 520 : 路由信息
    - tcp
      - 20 : FTP
      - 21 : FTP
      - 23 : TELNET
      - 25 : SMTP
      - 80 : HTTP
      - 119 : NNTP
      - 179 : BGP
  - 注册端口号(1024-49151)
  - 临时端口号(49152-65535)
- UDP数据报文格式
  - UDP报头(64 bit)
    - 源端口号(16 bit)
    - 目的端口号(16 bit)
    - UDP总长度(16 bit)
    - 校验和(16 bit)
  - 数据(不足16 bit 按16bit扩展)
- TCP数据报文格式
  - TCP报头(20 - 60 byte)
    - 源端口号(16 bit)
    - 目的端口号(16 bit)
    - 发送序号(32 bit)
    - 确认序号(32 bit)
    - 报头长度(4 bit)
    - 保留(6 bit)
    - URG(1 bit)
    - ACK(1 bit)
    - PSH(1 bit)
    - RST(1 bit)
    - SYN(1 bit)
    - FIN(1 bit)
    - 窗口(16 bit)
    - 校验和(16 bit)
    - 紧急指针(16 bit)
    - 选项及填充( =< 40byte)
  - 数据
