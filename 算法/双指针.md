## 快慢指针
> LC 26. 给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度
```java
// O(1)
public int removeDuplicates(int[] nums) {
    if (nums.length == 0 ) return 0;
    int slow = 0;
    for (int quick = 1; quick < nums.length; quick++) {
        if (nums[slow] != nums[quick]) {
            nums[++slow] = nums[quick];
        }
    }
    return ++slow;
}
```
> LC 19. 删除链表的倒数第N个节点
```java
// 链表一般头部添加 哑节点（dummy node）
// 快慢指针相差N, 所以快指针到末尾慢节点为虚删除节点
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode mock = new ListNode();
    mock.next = head;
    ListNode slow = mock;
    ListNode fast = mock;
    for (int i = 0; i < n; i++) {
        fast = fast.next;
    }
    for (; fast.next != null;) {
        slow = slow.next;
        fast = fast.next;
    }
    slow.next = slow.next.next;
    return mock.next;
}
```
> LC 876.给定一个带有头结点head的非空单链表，返回链表的中间结点
```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}
```

> LC 160. 编写一个程序，找到两个单链表相交的起始节点
```java
// 指针A: A + B
// 指针B: B + A
// 如果 A , B 相交于某点, 于某点  指针A == 指针B
public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
    if (headA == null || headB == null) return null;
    ListNode nodeA = headA;
    ListNode nodeB = headB;
    boolean aa = false;
    boolean bb = false;
    while (true) {
        if (nodeA == nodeB) return nodeA;
        nodeA = nodeA.next;
        if (nodeA == null) {
            if (!aa) {
                aa = true;
                nodeA = headB;
            } else {
                break;
            }
        }
        nodeB = nodeB.next;
        if (nodeB == null) {
            if (!bb) {
                bb = true;
                nodeB = headA;
            }
            
        }
    }
    return null;
}
```
## 对撞指针
> LC 344. 编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 char[] 的形式给出
```java
void reverseString(char[] s) {
    int n = s.length;
    for (int left = 0, right = n - 1; left < right; ++left, --right) {
        swap(s[left], s[right]);
    }
}
```
> LC 977. 给定一个按非递减顺序排序的整数数组A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序
```java
public int[] sortedSquares(int[] A) {
    if (A[0] >= 0) {
        for (int i = 0; i < A.length; i++) {
            A[i] = A[i] * A[i];
        }
        return A;
    }
    int[] B = new int[A.length];
    int curIdx = A.length - 1;
    for(int left =0, right = A.length - 1; left <= right;) {
        if (A[left] >= 0) {
            B[curIdx] = A[right] * A[right];
            right--;
        } else {
            int a = A[left] * A[left];
            int b = A[right] * A[right];
            if (a > b) {
                B[curIdx] = a;
                left++;
            } else {
                B[curIdx] = b;
                right--;
            }
        }
        curIdx--;
    }
    return B;
}
```
