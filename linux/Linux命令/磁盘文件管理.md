# 磁盘管理
- du : 统计文件和目录占用的磁盘空间
    - a, --all
    - c, --total
    - h, --human-readable
- df : 显示文件系统磁盘使用情况
    - a, --all
    - h, --human-readable
    - i, --inodes
- mkswap : 设置交换分区
    - device
- mount : 挂载指定文件系统
    - mount /dev/fd0 /mnt/floppy
    - mount -t iso9660 /dev/cdrom /mnt/cdrom
    - mount -t vfat /dev/sdb /mnt/flash
- umount : 卸载挂载的文件系统
